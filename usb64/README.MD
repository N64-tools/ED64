# PC Program for communicating with the ED64 via USB

## supported commands:
### FPGA
Loads an RBF file into the ED64's fpga
Arg: -fpga <filename>


### Transfer ROM
Writes a (ROM) file to ED64's volatile memory
Arg: -rom <filename>


### Start ROM
Starts the ROM written to the ED64's volatile memory
Arg: -start
